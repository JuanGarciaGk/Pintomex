Descripción General del Sistema
El sistema web permitirá gestionar de manera eficiente las operaciones comerciales de la empresa a través de las siguientes funcionalidades:

Funcionalidades Principales
•	Registro de ventas mediante escaneo de código de barras.
•	Cálculo automático de subtotal, impuestos y total.
•	Actualización automática del inventario en tiempo real.
•	Administración de productos (crear, editar y eliminar).
•	Control de entradas y salidas de mercancía.
•	Generación de reportes de ventas y estadísticas.
•	Almacenamiento de operaciones en base de datos.
•	Emisión de tickets de venta.

Tecnologías Utilizadas
El sistema será desarrollado utilizando tecnologías web tradicionales, trabajando en entorno local con XAMPP.
•	HTML: Estructura del sitio web.
•	CSS: Diseño visual e interfaz gráfica.
•	JavaScript: Interactividad y dinamismo.
•	PHP: Lógica del lado del servidor.
•	MySQL (incluido en XAMPP): Base de datos relacional.
•	Apache (incluido en XAMPP): Servidor web.
